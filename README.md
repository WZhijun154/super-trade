你自己搭一套全自动量化系统，需要做什么？

第一步：数据层（最费时间）

日线、1分钟线、Tick数据。全市场5182只股票。每天自动更新。不依赖聚宽/米筐，自己用QMT+akshare存到本地parquet。

这一步我花了2个多月才跑通。大部分人在这一步就放弃了。

第二步：回测层（最烧脑）

写一个能跑全成本逐日逐笔复利的回测引擎。不是简单的pandas运算——要考虑：资金约束、滑点、手续费、印花税、被迫清仓、多策略资金竞争。

我跑了17年数据，最终只有恐慌反转和低波防御两个方向通过了验证。

第三步：执行层（最怕出bug）

QMT接入，10秒扫描一次全市场，信号来了自动买入，触发止损自动卖出，每日报告自动生成。

我现在8个策略并行跑模拟盘，全部无人值守，每天收盘看报告就行。

这三层全部跑通的人，国内真的不多。我自己也是靠着AI Agent辅助才做到的——策略决策是我做的，代码实现交给AI。
